# نظام إدارة الاقتراع (VMS)

موقع كامل: لوحة تحكم، إدارة ناخبين، **اقتراع سريع** بضغطة واحدة، استيراد/تصدير Excel، صلاحيات (مدير / موظف)، وسجل عمليات.

## التشغيل السريع

```bash
npm install
cd server && npm install && cd ../client && npm install
cd ..
npm run dev
```

- الواجهة: [http://localhost:5173](http://localhost:5173)
- الـ API: [http://localhost:4000/api/health](http://localhost:4000/api/health)

### حسابات جاهزة (أول تشغيل)

| المستخدم | كلمة المرور | الدور   |
|----------|-------------|---------|
| admin    | admin123    | مدير    |
| staff    | staff123    | موظف    |

غيّر `JWT_SECRET` في `server/.env` قبل أي استخدام حقيقي.

## بناء للإنتاج (خادم واحد يخدم الواجهة + API)

```bash
cd client && npm run build && cd ../server && npm run build && cd ..
npm start
```

ثم افتح `http://localhost:4000` (يُخدم ملفات `client/dist` إن وُجدت).

## استيراد Excel

الصف الأول: أعمدة مقترحة `full_name`, `national_id`, `area` (اختياري)، ويمكن `status` و `voted_at` للحالات الخاصة. يدعم أيضاً أسماء عربية للأعمدة كما في الواجهة.

## QR للصندوق

رابط يفتح الاقتراع السريع مع الرقم التعريفي (يحتاج تسجيل دخول):

`/quick?nid=الرقم_التعريفي`

## المتطلبات

- **Node.js 22.16+** (يُفضّل 24+) لأن الخادم يستخدم **`node:sqlite`** المدمج دون تجميع C++.

## التقنيات

- React (Vite) + Tailwind
- Node.js + Express + SQLite (`node:sqlite`)
- JWT للجلسات

قاعدة البيانات في `server/data/app.db` (تُنشأ تلقائياً).

## بيانات تجريبية (اختياري)

```bash
cd server && npx tsx scripts/seed-demo.ts
```

يضيف 500 ناخباً تجريبياً (يمكنك تعديل العدد داخل الملف).

## دفعات Excel (مدرسة / ملف)

- من الواجهة (مدير): **دفعات Excel** — إنشاء دفعة بعنوان (مثلاً اسم المدرسة)، حذف دفعة **يحذف كل الناخبين** المرتبطين بها.
- شريط **الكل / اسم الدفعة** أعلى الموقع يفلتر لوحة التحكم والناخبين والاقتراع السريع **دون** فتح موقعين.
- **استيراد Excel** من صفحة الناخبين يتطلب اختيار دفعة أولاً.

## استيراد من سطر الأوامر

```bash
cd server
npx tsx scripts/import-excel-file.ts "C:\...\ملف.xlsx" --title="مدرسة صبحة"
# أو لدفعة موجودة:
npx tsx scripts/import-excel-file.ts "C:\...\ملف.xlsx" --batch-id=2
```

- `--wipe`: يحذف كل الناخبين **وجميع الدفعات** ثم يستورد (بحذر شديد).

الأعمدة: `مركز التسجيل والاقتراع`، `رمز الناخب`، `#` (رقم الصف في الجدول)، وحقول الاسم الأربعة.

## خلفية الكوفية (صورة اختيارية)

ضع ملف الصورة باسم `keffiyeh-bg.png` داخل المجلد `client/public/` ثم أعد بناء الواجهة. بدون الملف تبقى خلفية الشبكة الافتراضية.

## تحديث عمود # من Excel (دون إعادة استيراد كامل)

من الواجهة (مدير): في صفحة الناخبين زر **«تحديث # من ملف»** — نفس أعمدة الملف الأصلية (رمز الناخب + #). أو عبر API: `POST /api/voters/sync-row-numbers` مع `multipart/form-data` وحقل `file`.
